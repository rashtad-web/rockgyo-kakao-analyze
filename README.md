# 💬 카카오톡 채팅 통계 분석기

카카오톡 채팅 내보내기 파일을 분석하여 다양한 통계를 제공하는 웹 애플리케이션입니다.

## 📋 주요 기능

- **파일 업로드**: 카카오톡 채팅 내보내기 파일(.txt) 업로드 및 분석
- **날짜/시간 필터링**: 초 단위까지 정밀한 기간 필터링
- **커스텀 키워드 분석**: 사용자 정의 키워드로 채팅 내용 분석
- **다양한 통계 제공**: 참여자, 시간대, 메시지 타입, 감정 분석 등

## 📊 제공 통계

### 기본 통계
- 총 메시지 수
- 총 참여자 수
- 메시지 타입별 통계 (텍스트, 사진, 동영상, 이모티콘, 링크)

### 참여자 분석
- 상위 참여자 (Top 20)
- 평균 메시지 길이별 참여자
- 멘션 횟수별 참여자
- 울음/웃음 표현 사용자

### 시간대 분석
- 시간대별 메시지 분포
- 요일별 메시지 분포
- 날짜별 메시지 분포
- 심야 메시지 (새벽 2-5시) 통계

### 메시지 패턴
- 자주 사용되는 단어 (Top 20)
- 연속 메시지 최고 기록 (5분 이내 연속 메시지)
- 사진/동영상/링크 공유 통계

### 키워드 분석
- 커스텀 키워드별 언급 횟수
- 키워드별 가장 많이 언급한 사람

## 🆕 v1.1 변경 사항

### ✨ 새로 추가된 통계

#### 1. 대화 주도자 & 종료자
- **대화 주도자**: 이전 메시지와 1시간 이상 간격이 있을 때 대화를 시작한 사람 (Top 20)
- **대화 종료자**: 메시지 후 1시간 동안 아무도 메시지를 보내지 않아 대화를 종료한 사람 (Top 20)

#### 2. 감정/표현 분석
- **긍정 표현**: "좋아", "최고", "고마워", "사랑", "행복" 등 (Top 20)
- **부정 표현**: "싫어", "안돼", "아니", "화나", "슬프" 등 (Top 20)
- **질문 표현**: "?", "뭐", "어디", "언제", "누구", "왜" 등 (Top 20)
- **감탄사**: "와", "헐", "대박", "와우" 등 (Top 20)

#### 3. 시간대별 활동 패턴
- **새벽** (0-5시)
- **아침** (6-11시)
- **점심** (12-13시)
- **오후** (14-17시)
- **저녁** (18-21시)
- **밤** (22-23시)

각 시간대별 메시지 수와 비율을 표시합니다.

#### 4. 메시지 길이 패턴
- **한 줄 메시지**: 줄바꿈이 없는 메시지
- **짧은 메시지**: 5자 이하
- **중간 메시지**: 6-50자
- **긴 메시지**: 51-100자
- **매우 긴 메시지**: 100자 이상

참여자별 메시지 길이 패턴도 함께 제공됩니다 (Top 20).

#### 5. 대화 밀도 분석
- **하루 평균 메시지 수**: 전체 기간 동안 하루 평균 메시지 수
- **활동한 날**: 메시지가 있었던 날짜 수와 비율
- **가장 활발한 날**: 메시지가 가장 많았던 날짜와 메시지 수
- **가장 조용한 날**: 메시지가 가장 적었던 날짜와 메시지 수
- **가장 긴 공백 기간**: 메시지가 없었던 가장 긴 기간 (일 단위)

### 🔧 개선 사항

#### 연속 메시지 설명 명확화
- "연속 메시지(스팸) 최고 기록" → "연속 메시지 최고 기록"으로 제목 변경
- 설명 추가: "같은 사람이 5분 이내에 연속으로 보낸 메시지의 최대 개수입니다."
- 연속 메시지는 같은 메시지를 반복한 것이 아니라, 서로 다른 메시지를 5분 이내에 연속으로 보낸 경우를 의미합니다.

#### 제거된 통계
- 가장 긴 메시지 통계 제거
- 가장 짧은 메시지 통계 제거

## 🆕 v1.2 변경 사항

### ✨ 새로 추가된 통계

#### 1. 참여자 간 상호작용 (멘션 관계도)
- **멘션 관계도**: 누가 누구를 가장 많이 멘션했는지 Top 30 표시
- 멘션한 사람 → 멘션당한 사람 관계를 테이블로 표시
- 참여자 간 상호작용 패턴을 한눈에 파악 가능

#### 2. 메시지 타임라인 히트맵
- **월별 메시지 수**: 각 월별 총 메시지 수 추이
- **주간별 메시지 수**: 각 주의 시작일 기준 메시지 수 추이
- 시간에 따른 대화 활동 패턴을 시각적으로 확인 가능

#### 3. 대화 참여도 변화 추이
- **월별 참여자 수**: 각 월에 활동한 참여자 수 변화
- **참여자별 활동 기간** (Top 20): 각 참여자의 첫 메시지, 마지막 메시지, 총 활동 기간(일)
- 대화방의 참여도 변화와 각 참여자의 활동 기간을 한눈에 확인

### 🔧 개선 사항

#### UI 개선
- **테이블 헤더 가운데 정렬**: 모든 테이블의 헤더를 가운데 정렬하여 가독성 향상
- **순위 열 개선**: 순위 열의 너비를 확대하고 개행 방지 처리
- **테이블 세로 구분선 추가**: "참여자별 메시지 길이 패턴" 등 테이블에 세로 구분선 추가로 항목 구분 용이

#### 소셜 링크 추가
- **GitHub 저장소 링크**: 좌측 상단에 원형 GitHub 버튼 추가 (다른 SNS 로고와 구분되는 디자인)
- **기본 키워드 업데이트**: 기본 키워드에 "락교" 추가

## 🚀 사용 방법

1. 카카오톡에서 채팅방 설정 → 대화 내용 내보내기
2. 내보낸 `.txt` 파일을 선택
3. 원하는 기간과 키워드를 설정 (선택 사항)
4. "적용" 버튼을 클릭하여 통계 확인

## 🛠️ 기술 스택

- **Frontend**: HTML, CSS, TypeScript
- **Build Tool**: TypeScript Compiler
- **Package Manager**: Yarn
- **Server**: http-server

## 📦 설치 및 실행

```bash
# 의존성 설치
yarn install

# 빌드
yarn build

# 개발 서버 실행
yarn serve
```

서버는 기본적으로 `http://localhost:3000`에서 실행됩니다.
